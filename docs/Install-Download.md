## GIT download

GitHUB hosts the main repository for OpenSIPS. In order to checkout the latest version of OpenSIPS, you can run:

```text
git clone --recurse-submodules https://github.com/OpenSIPS/opensips.git -b 4.0 opensips-4.0
```
